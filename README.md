👔 VELVORÉ - Premium Men's Apparel Landing Page

A modern, responsive, and visually appealing fashion landing page designed for a premium men's apparel brand. Built using HTML, CSS, and JavaScript, this project showcases luxury fashion collections through elegant layouts, smooth animations, interactive elements, and a professional user experience.

✨ Features

* Modern Glassmorphism Navigation Bar
* Responsive Design for All Devices
* Luxury Fashion Hero Section
* Interactive Scroll Progress Bar
* Smooth Scrolling Navigation
* Animated Section Reveal Effects
* Premium Product Collection Showcase
* Hover Animations & Micro-Interactions
* Modern Dark Theme UI
* Mobile-Friendly Layout
* Clean and Professional Design

🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* CSS Grid
* Flexbox
* Google Fonts (Poppins)
* Responsive Web Design
* DOM Manipulation
